# 📝 Task Manager Web Application

## 📌 Description

This is a full-stack Task Manager application built using **Spring Boot** for backend and **HTML, CSS, JavaScript** for frontend.
It allows users to manage projects and tasks easily.

---

## 🚀 Features

* 🔐 User Login (JWT Authentication)
* 📁 Create and manage projects
* ✅ Add tasks
* ✔ Mark tasks as completed
* ❌ Delete tasks
* 📊 Dashboard showing total, pending, and completed tasks
* ☁ Deployed on Railway
* 🗄 Connected to MongoDB Atlas

---

## 🛠 Technologies Used

* **Backend:** Spring Boot (Java)
* **Frontend:** HTML, CSS, JavaScript
* **Database:** MongoDB Atlas
* **Deployment:** Railway
* **Version Control:** Git & GitHub

## ⚙️ How to Run Locally

1. Clone the repository

   ```bash
   git clone https://github.com/kalyanimaturi/task-manager
   ```

2. Open project in IDE (Eclipse/IntelliJ)

3. Configure MongoDB connection in `application.properties`

4. Run Spring Boot application

5. Open browser:

   ```
   http://localhost:8081
   ```
